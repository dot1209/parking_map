## Getting Started
### Built with
* Bootstrap 5
* FastAPI
* jQuery

### Prerequisites
* [python](https://www.python.org/downloads/)
> Recommend using python 3.8

### Installation
1. Clone the repo
```
git clone https://github.com/dot1209/parking_map.git
```
2. Install python packages <br />
```
pip install -r requirements.txt
```

### Usage
```
cd app 
uvicorn app:app --reload
```
